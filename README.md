## MHXX Copium Mod
A version of the original MHXX Cope Mod made by Aradi with extra changes.

Credits\Thanks:

Aradi147 for originally making Cope Mod. https://github.com/Aradi147/MHGUCope/tree/main

Vaccaria for fixing Blast Dash in the original Cope Mod.

## How to install the Copium Mod:

1- Download the latest release available

2- Open your 3ds emulator, right click the game, hover to open then mods location 
(you can also look for the directory in `your3dsEmu/load/mods/0004000000197100`)

3- Extract the romfs to that folder

4- You're good to go. Have fun!
